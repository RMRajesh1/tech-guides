00 Introduction

About BlenderMCP

BlenderMCP is an open-source project by Siddharth Ahuja that implements the Model Context Protocol to let AI assistants directly control Blender. 17K+ GitHub stars, active Discord community. Not an official Blender product.

🚫

No official website. Any site claiming to be official is a fake. Only trust github.com/ahujasid/blender-mcp.

How It Works

Two processes communicate over a local TCP socket. The AI client auto-starts the MCP server — you only click one button in Blender.

02 Setup — Step 2

Installing the Blender Addon

addon.py opens a TCP socket server inside Blender so the MCP server can send commands.

Step 1 — Download addon.py

macOS / Linux
Terminal
$ curl -L "https://github.com/ahujasid/blender-mcp/raw/main/addon.py" -o ~/Downloads/addon.py
Linux
Terminal
$ curl -L "https://github.com/ahujasid/blender-mcp/raw/main/addon.py" -o ~/Downloads/addon.py
Windows
PowerShell
PS> Invoke-WebRequest -Uri "https://github.com/ahujasid/blender-mcp/raw/main/addon.py" -OutFile "$env:USERPROFILE\Downloads\addon.py"

Or just open the link in your browser and save: github.com/ahujasid/blender-mcp/raw/main/addon.py

Step 2 — Install in Blender

1
Open Blender Preferences

Edit → Preferences → Add-ons tab

2
Install the file

Blender 3.x – 4.1: Click Install… → select addon.py

Blender 4.2+: Dropdown arrow (top-right) → Install from Disk… → select addon.py

3
Enable it

Search "MCP" → check the box next to Interface: Blender MCP

ℹ️

Updating? Re-download latest addon.py, reinstall in Blender, then delete and re-add the blender entry in your AI client config.

Starting the Connection

1

🚫

Do NOT run uvx blender-mcp manually. The AI client starts it automatically. Running it manually causes conflicts.

⚠️

One client at a time only. Don't run Claude Desktop and Cursor simultaneously against the same Blender session.

04 Configuration

Environment Variables

BLENDER_HOSTHost address. Default: localhost. Use host.docker.internal for Docker.
BLENDER_PORTPort for the socket server. Default: 9876.
DISABLE_TELEMETRYSet to true to disable all telemetry.

Add under the "env" key in any JSON client config:

Any JSON client — env vars exampleJSON
{
 "mcpServers": {
  "blender": {
   "command": "uvx", "args": ["blender-mcp"],
   "env": {
    "BLENDER_PORT": "9876",
    "DISABLE_TELEMETRY": "true"
   }
  }
 }
}

02
Materials from Natural Language
Describe appearance, get accurate Principled BSDF setups
BEGINNER
🗣 Prompt
"Make this car red and metallic."
execute_blender_codepython
import bpy
mat = bpy.data.materials.new("MetallicRed")
mat.use_nodes = True
bsdf = mat.node_tree.nodes["Principled BSDF"]
bsdf.inputs["Base Color"].default_value = (0.8, 0.02, 0.02, 1.0)
bsdf.inputs["Metallic"].default_value  = 0.9
bsdf.inputs["Roughness"].default_value = 0.2
bpy.data.objects["Car"].data.materials.append(mat)
✅ Result
  • Car turns metallic red immediately
  • Metallic=0.9 gives realistic paintwork sheen
03
Procedural Python Scripting
Loops, randomization, parametric geometry
ADVANCED
🗣 Prompt
"Create a 5×5 grid of cubes, each with a randomized height between 0.5 and 3 units."
execute_blender_codepython
import bpy, random
bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete()
for x in range(5):
  for y in range(5):
    h = random.uniform(0.5, 3.0)
    bpy.ops.mesh.primitive_cube_add(location=(x*1.2, y*1.2, h/2))
    bpy.context.active_object.scale = (0.4, 0.4, h/2)
    bpy.context.active_object.name = f"Column_{x}_{y}"
✅ Result
  • 25 columns in a 5×5 grid with city skyline effect
  • Each uniquely named in Outliner
04
Camera & Lighting
Isometric camera, studio three-point lighting
INTERMEDIATE

import bpy, math
cam = bpy.data.objects["Camera"]
cam.location = (7, -7, 7)
cam.rotation_euler = (math.radians(60), 0, math.radians(45))
cam.data.type = 'ORTHO'; cam.data.ortho_scale = 8
for name, loc, energy in [("Key",(4,-4,6),1000),("Fill",(-4,-2,4),300),("Back",(0,5,5),200)]:
  bpy.ops.object.light_add(type='AREA', location=loc)
  bpy.context.active_object.data.energy = energy
✅ Result

13 Help

Troubleshooting

Click to expand. Issues ordered by how often they occur.

Wrong Python version — MCP server won't start
PYTHON 3.12 ONLY

BlenderMCP requires exactly Python 3.12. Running under 3.11 or 3.13+ causes silent failures. Most common cause of "tools not loading."

macOS / Linux
Check & fix
$ python3 --version
Python 3.12.x ← correct
Python 3.11.x ← wrong, will fail

# Install 3.12 via uv
$ uv python install 3.12
Linux
Check & fix
$ python3 --version
Python 3.12.x ← correct
$ uv python install 3.12
Windows
PowerShell
PS> python --version
Python 3.12.x ← correct
PS> uv python install 3.12

Pin 3.12 in your config args: "args": ["--python", "3.12", "blender-mcp"]

📄claude_desktop_config.json — pinned Python 3.12JSON
{
 "mcpServers": {
  "blender": {
   "command": "uvx",
   "args": ["--python", "3.12", "blender-mcp"]
  }
 }
}

MCP server doesn't appear after config
CONFIG

Validate config syntax — a single bad character breaks everything silently.

macOS / Linux
Validate JSON config
$ python3 -m json.tool ~/Library/Application\ Support/Claude/claude_desktop_config.json
Linux
Validate JSON

Windows
PowerShell
PS> Get-Content "$env:APPDATA\Claude\claude_desktop_config.json" | ConvertFrom-Json

Common mistakes: trailing commas in JSON; config.yaml instead of config.toml for Codex; JSON syntax inside TOML. After fixing, fully quit and relaunch your AI client.

Timeout or complex request fails mid-way
TIMEOUT

Break requests into smaller steps: "Add the keep" → "Add outer walls" → "Apply stone texture."

From the README: "Have you tried turning it off and on again? Restart both Claude and the Blender server."

execute_blender_code — Blender crashes or hangs
SAFETY

⌘ S / Ctrl S — save your .blend file before every AI session. Enable auto-save: Edit → Preferences → Save & Load.

From the README: "execute_blender_code runs arbitrary Python — powerful but potentially dangerous. ALWAYS save your work."
